## ✨ About ChikAI

**ChikAI** is a modern, minimalist AI chatbot that combines the Filipino word "Chika" (meaning story/chat) with AI technology. It's your friendly **Chismosa AI** companion that loves to have chika and engage in warm, conversational Taglish (Tagalog + English) exchanges.

> 💅 *"Chismis time! What's the tea? ☕"*

### 🌟 Features

- 🤖 **AI-Powered Conversations** - Powered by Groq's Llama 3.3 70B model for fast, intelligent responses
- 🌓 **Dark/Light Mode** - Seamless theme switching with persistent preference
- 📱 **Responsive Design** - Beautiful on desktop, tablet, and mobile
- ⚡ **Real-time Streaming** - See responses as they're generated
- 🎨 **Modern Minimalist UI** - Clean, elegant, and focused on conversation
- 🇵🇭 **Taglish Support** - Responds in a mix of English and Tagalog
- 🔒 **Secure API Integration** - API keys kept safe on the server
- 🚀 **Production Ready** - Deployed on Vercel with automatic deployments

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) with App Router
- **Frontend**: [React 18](https://reactjs.org/) + [Tailwind CSS](https://tailwindcss.com/)
- **AI Provider**: [Groq](https://groq.com/) (Llama 3.3 70B)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)
- **Language**: TypeScript

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn
- A [Groq API key](https://console.groq.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ymm-dev/chikai.git
   cd chikai
   
2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   
3. **Set up environment variables**
   ```bash
   #Create .env file in your root folder and paste your groq API Key
   GROQ_API_KEY=your_groq_api_key_here

4. . **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
